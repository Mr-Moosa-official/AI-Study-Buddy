# **App Name**: AI Study Buddy

## Core Features:

- Daily Plan Generation: Generate a personalized daily study plan based on upcoming deadlines, subject difficulty, and past performance.
- Progress Tracking: Track study progress by marking topics as complete and recording practice test scores.
- Adaptive Planning: Adjust the study plan in real-time based on tracked progress, performance on practice tests, and approaching deadlines. AI analyzes performance to focus on weak areas.
- Reminder System: Set up reminders for upcoming study sessions, practice tests, and assignment deadlines.
- AI-Powered Resource Recommendation: The app acts as an intelligent tool to determine relevant study materials from web links or shared documents that will best reinforce user knowledge.

## Style Guidelines:

- Primary color: Calming blue (#64B5F6) to promote focus and concentration.
- Background color: Light, desaturated blue (#E3F2FD) for a clean and unobtrusive backdrop.
- Accent color: Energetic orange (#FFB74D) for highlighting key actions and important notifications.
- Body and headline font: 'PT Sans', a humanist sans-serif providing both modern and approachable feel.
- Use clear, minimalist icons to represent subjects, tasks, and features.
- Clean and intuitive layout with a focus on easy navigation and readability.
- Subtle animations to provide feedback and enhance the user experience.
